# Landing Page – MKV Luxury

## Objectif du projet

Ce projet consiste à créer une landing page dédiée afin de proposer
des améliorations et une refonte potentielle du site mkvluxury.com.
L’objectif principal de cette page est d’inciter le client à prendre
contact pour discuter de son projet digital.

---

## Analyse du site existant

Le site MKV Luxury présente une offre haut de gamme de location de
voitures de luxe. Le design est élégant et le catalogue est riche.
Cependant, le site peut être amélioré sur les points suivants :
- meilleure clarté du message principal,
- optimisation de l’expérience utilisateur,
- augmentation du taux de conversion via des pages dédiées.

---

## Pourquoi une landing page dédiée

Une landing page permet de se concentrer sur un objectif unique :
la prise de contact. Contrairement à un site vitrine classique,
elle guide l’utilisateur vers une action précise et améliore
la conversion des visiteurs en prospects.

---

## Intégration technique

La landing page intègre un formulaire de contact connecté à une
API mock simulant l’envoi des données côté serveur. Cela permet
de démontrer la capacité à relier un front-end à un service externe.

---

## Prochaines évolutions possibles

Avec plus de temps, il serait possible d’ajouter :
- un backend réel (FastAPI),
- une base de données pour stocker les messages,
- un tableau de bord administrateur,
- des statistiques de conversion,
- une optimisation mobile et performance plus poussée.

---

## Améliorations futures

Avec plus d’expérience, l’UX, le design et la sécurité pourraient
être davantage optimisés afin de proposer une solution encore
plus professionnelle et scalable.
